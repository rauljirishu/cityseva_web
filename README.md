# CitySeva - From Complaints to Care, Instantly

CitySeva is a full-stack civic complaint management mobile application 
designed for Indian citizens to report and track local civic issues 
directly to government authorities in real time.

## Overview

CitySeva bridges the gap between citizens and government by providing 
a transparent, efficient and accountable platform for civic issue 
resolution. Citizens can submit complaints, track their status and 
rate the government's performance — all from a single mobile application.

## Features

- Submit civic complaints with photo evidence and GPS location
- AI-powered complaint validity analysis using Google Gemini
- Real-time complaint status tracking and notifications
- Government performance analytics dashboard
- Authority complaint verification and assignment workflow
- Citizen feedback and satisfaction rating system
- Built-in user manual and step-by-step onboarding guide
- Offline support with local data caching

## Technology Stack

- **Frontend**  : Flutter (Web, Android, iOS)
- **Backend**   : Node.js with Express
- **Database**  : MongoDB Atlas (Free Tier)
- **AI**        : Google Gemini API

## Purpose

This application was developed as a solution to improve civic 
governance by making the complaint resolution process transparent, 
fast and data-driven. It empowers citizens with a voice and holds 
government officials accountable through performance metrics and 
citizen feedback.
